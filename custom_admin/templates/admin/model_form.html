Dropdown Configuration Guide
How to Configure Dropdowns:
  1. Field Type: Set to SELECT or MULTISELECT
  2. Source Entity: Choose the entity type that contains the data (e.g., Vessel, Customer)
  3. Source Attribute: Choose the column that provides the actual values
  4. Display Attribute: Optional - choose a different column for user-friendly display
  5. Unique Values Only: Check to show only distinct values
Example: For a "Vessel Name" dropdown:
  • Source Entity: Vessel (master)
  • Source Attribute: VESSEL_NAME
  • Display Attribute: VESSEL_NAME (same)
  • Unique Values: Yes
